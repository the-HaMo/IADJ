\section{Introducción}

\begin{justify}
 En esta memoria se documenta el diseño y la implementación del \textbf{Bloque 1: Movimiento} de la asignatura Inteligencia Artificial para Videojuegos. El propósito principal de este bloque es el desarrollo de un sistema de navegación reactivo y autónomo  (steering behaviors) para diferentes tipos de agentes dentro del motor de desarrollo \textbf{Unity}.
\newline
A lo largo de este documento, se detallarán los métodos y soluciones aplicados para resolver los desafíos de movilidad planteados en el enunciado. Se explicará cómo se han estructurado los elementos pedidos en el enunciado, algunos elementos fundamentales son los siguientes;

\begin{itemize}
\item \textbf{Motor de físicas: }Como diseñamos distintos aspectos de nuestro motor; como el movimiento angular, la aceleración, el movimiento rectilíneo, etc. Mediante las ecuaciones de Newton-Euler, operaciones de vectores, transformadas, etc. Con el objetivo de trabajar sobre una dimensión 2.5D.   
\item \textbf{Sistemas de Steering :} Desarrollo de algoritmos para el control de la aceleración y dirección de los agentes, incluyendo comportamientos básicos como Seek y Arrive, y delegados como Wander.
\item \textbf{Detección y Evitación de Obstáculos:} Implementación de sistemas de colisión basados en rayos (\textit{raycasts}) o "bigotes" para garantizar una navegación fluida en entornos complejos.
\item \textbf{Gestión de Grupos y Formaciones:} Organización de múltiples unidades bajo estructuras de movimiento coordinado y seguimiento de líderes.
\item\textbf{Búsqueda de Caminos (Pathfinding):} Aplicación del algoritmo \textbf{LRTA}* sobre una representación del escenario basada en rejillas (\textit{grids}) para alcanzar el objetivo.
\end{itemize}
\end{justify}
